# PawStay Network //CloudSwam Project

A modern responsive website for a House Sitting & Pet Boarding Network. The project is built with HTML5, CSS3, Tailwind via CDN, Bootstrap components, and vanilla JavaScript.

## Features

- Fully responsive layout for desktop, tablet, and mobile
- Light mode and dark mode with saved user preference
- Responsive navigation bar with mobile menu
- Hero section with strong call-to-action buttons
- Services, why choose us, testimonials, FAQ, pricing, and contact sections
- Bootstrap accordion and form components
- Smooth hover effects, transitions, and scroll reveal animations
- Contact and sitter application form validation using vanilla JavaScript
- Clean folder organization for HTML, CSS, JavaScript, and assets

## Pages

- `index.html` - Home
- `pages/about.html` - About

## How to Run

Open `index.html` in a browser.

For a local server preview, run one of the following commands from the project folder:

```bash
python -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## JavaScript Functionality

The `js/script.js` file handles:

- Dark/light theme switching
- Theme preference persistence with `localStorage`
- Mobile navigation toggle
- Active navigation state
- Scroll reveal animations
- Basic form validation and success messages
